# 围棋教学网站前端
>vue.js+Element ui

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

## 说明
教学网站共分为四个部分：对弈、教学、练习和测评

* 对弈：
